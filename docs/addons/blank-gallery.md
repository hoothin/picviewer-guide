# Blank Gallery

The Blank Gallery Page is used to open the gallery independently, suitable for local or cross-site organization.

## How to Open

Visit: https://hoothin.github.io/UserScripts/Picviewer%20CE+/gallery.html

Directly open in browser, then drag and drop:
- Images
- Folders
- Video or Audio

## URL Parameters

| Parameter | Description |
| --- | --- |
| `mode` | `1` for ViewMore mode |
| `imgs` | Image URL list (space separated) |

### Example

```
https://hoothin.github.io/UserScripts/Picviewer%20CE+/gallery.html?mode=0&imgs=http://xxx/xxx[01-99].jpg
```

> Supports `[01-99]` numeric range expansion.
