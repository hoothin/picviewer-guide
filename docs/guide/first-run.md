# First Run

::: tip Congrats on your installation
You're ready to go. Below are the best next steps to get productive fast.
:::

## Next steps

1. **Verify it's working**: Refresh any webpage, hover an image, and confirm the float bar appears.
2. **Try the core flows**: Open Gallery with `Ctrl + G`, then test quick preview by holding `Ctrl` and hovering.
3. **Tune downloads**: In Settings, choose your preferred download method (ZIP or aria2) and default format.
4. **Optional addons**: If you need PDF export, install the PDF addon from the Addons section.
5. **Fix tricky sites**: If a site blocks originals or uses lazy loading, enable or add rules.
