# Open Settings

::: tip This page was opened due to page restrictions
The settings window will open on this page.
:::

## If the settings window still does not appear

1. Make sure your userscript manager is enabled.
2. Refresh this page once.
3. If you are using Chrome 138+ or similar, enable **Allow User Scripts** or **Developer Mode** for the userscript manager.
