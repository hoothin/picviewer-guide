## What Can You Do?

- Hover to preview any image with quick zoom, flip, rotate, and drag.
- One-click Gallery Mode to collect and manage page images in bulk.
- Use the Rule System to adapt to special sites and grab high-res images reliably.
- Search by image or batch download to build a complete image workflow.

> This documentation covers installation, configuration, rule writing, and practical examples, suitable for beginners to advanced rule maintainers.
