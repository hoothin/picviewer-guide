# PDFアドオン

PDFアドオンは、ZIPパッケージングをPDFデジタルアルバム生成に置き換えます。
画像コレクションの一括整理に適しています。

## ダウンロードリンク

- https://hoothin.github.io/UserScripts/Picviewer%20CE+/pvcep_pdf_addon.user.js

## 使用方法

1. PDFアドオンスクリプトをインストールします。
2. ギャラリーで「ZIPに圧縮」を有効にします。
3. エクスポート時にPDFファイルを直接生成します。

## 典型的なシナリオ

Webサイトの画像の番号が連続している場合、ギャラリーで一括追加してPDFとしてエクスポートできます。
例えば `1.jpg` から `99.jpg`。

## 注意事項

- アドオンは `jsPDF` に依存しているため、初回使用時は読み込みに時間がかかる場合があります。
- PDF生成プロセスは画像を1枚ずつ挿入するため、大量の場合はバッチ処理をお勧めします。
