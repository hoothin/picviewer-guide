# ダウンロードと形式変換

ダウンロード機能はギャラリーとフローティングバーに集中しており、複数のエクスポート方法をサポートしています。

## ダウンロード方法

- **直接ダウンロード**: ローカルストレージへの単一画像または一括ダウンロード。
- **ZIPパック**: ワンクリックでパッケージ化。大量のバッチ処理に適しています。
- **aria2リモート**: aria2 RPCにタスクを送信します。

## 形式変換

`gallery.formatConversion` を使用して変換ルールを定義できます。例：

```
webp>png
```

複数行の設定をサポートし、順に試行されます。

## aria2 設定

| 設定項目 | 説明 | デフォルト値 |
| --- | --- | --- |
| `gallery.aria2Host` | RPCアドレス | `http://localhost:6800` |
| `gallery.aria2Token` | RPCトークン | "" |

## ダウンロードペース制御

`gallery.downloadGap` は、リクエスト間隔を制御し、サイトのレート制限に引っかからないようにするために使用されます。

## 推奨される使用法

- 一括ダウンロードにはZIPまたはaria2を優先的に使用してください。
- 高解像度画像サイトでは、後の処理が不便になる可能性があるwebpファイルのダウンロードを避けるために、形式変換を使用することをお勧めします。
