# よくある質問

## フローティングバーが表示されない

考えられる原因と解決策：
- 画像が小さすぎる：`floatBar.forceShow.size` または `floatBar.minSizeLimit` を調整してください。
- 画像が背景にある：`floatBar.listenBg` を有効にしてください。
- 特殊なサイト構造：ルールシステムで `ext` または `getExtSrc` を追加してください。

## 元画像が取得できない

- `r/s` 置換または `getImage` ロジックが必要か確認してください。
- 内部ページへのアクセスが必要なサイトには `xhr` ルールを使用してください。
- 遅延読み込みサイトの場合は `lazyAttr` を指定してください。

## ギャラリーの画像が不完全

- `gallery.loadAll` または `gallery.viewmoreEndless` を有効にしてください。
- 「もっと読み込む」コマンドを使用してください。
- URLフィルタ条件が存在するか確認してください。

## ダウンロードの失敗またはタイムアウト

- `gallery.downloadGap` を適切に増やしてください。
- ブラウザのブロックを減らすためにaria2ダウンロードを試してください。
- 一部のサイトはクロスオリジンダウンロードを制限しています。代わりにリンクの一括コピーを試してください。

## 画像検索ポップアップが表示されない

- 対応するエンジンが有効になっているか確認してください。
- 設定で `firstEngine` を調整するか `gallery.searchData` を更新してください。
