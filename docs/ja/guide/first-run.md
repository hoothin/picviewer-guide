# 初回実行

::: tip インストール完了おめでとうございます
すぐに使い始められるよう、次の手順をまとめました。
:::

## 次のステップ

1. **動作確認**：任意のページを更新し、画像にカーソルを合わせてフロートバーが表示されるか確認します。
2. **コア機能を体験**：`Ctrl + G` でギャラリーを開き、`Ctrl` を押しながらホバーしてクイックプレビューを試します。
3. **ダウンロード設定**：設定でダウンロード方式（ZIP または aria2）と既定の形式を選びます。
4. **オプションのアドオン**：PDF 出力が必要ならアドオンページで PDF アドオンを導入します。
5. **サイト固有の調整**：縮小画像や遅延読み込みで原寸が取れない場合はルールを有効化または追加します。
