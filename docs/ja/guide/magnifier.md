# 拡大鏡

拡大鏡は局所的な拡大表示を提供し、詳細の確認やピクセルレベルの観察に適しています。

## 主要機能

- マウスの動きに合わせて拡大エリアを表示
- マウスホイールによるズームをサポート
- ズーム範囲とステップを設定可能

## 一般的な設定

| 設定項目 | 説明 | デフォルト値 |
| --- | --- | --- |
| `magnifier.radius` | 拡大鏡の半径 | `77` |
| `magnifier.wheelZoom.enabled` | ホイールズームを有効化 | `true` |
| `magnifier.wheelZoom.pauseFirst` | ズーム前に一時停止 | `true` |
| `magnifier.wheelZoom.scaleImage` | レンズ移動ではなく画像を拡大 | `true` |
| `magnifier.wheelZoom.range` | ズーム倍率リスト | `0.1 ~ 9.0` |

## 使用上のヒント

- 拡大鏡が敏感すぎる場合は、ホイールズームを無効にするか、倍率リストを調整してください。
- 移動追従が不便な場合は、「ズーム前に一時停止」を一時的に有効にすることができます。
