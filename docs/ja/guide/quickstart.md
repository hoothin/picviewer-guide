# クイックスタート

## 1. ホバープレビュー

画像にマウスを合わせると、フローティングバーが表示されます。デフォルトでは、元画像、現在の画像、ギャラリー、拡大鏡、ダウンロードが含まれます。

## 2. ギャラリーを開く

`Ctrl + G` を押すと、ギャラリーモードに素早くアクセスできます。ギャラリーは現在のページの認識可能な画像を自動的に収集し、フィルタリング、並べ替え、ダウンロード機能を提供します。

## 3. クイックプレビュー

`Ctrl` を押しながら画像やリンクにホバーすると、大きな画像を直接プレビューできます。ターゲットサイトが重い場合は、安定性を向上させるために画像の読み込み待ちを有効にすることをお勧めします。

## 4. 一括ダウンロード

ギャラリーでは、以下を選択できます：

- ZIPとしてエクスポート
- 画像リストを直接ダウンロード
- aria2に送信

## 5. ルール適応

サムネイルや遅延読み込みにより元画像が取得できない場合は、ルールを追加または有効にしてください。ルールシステムは、URL置換、内部ページ取得、カスタムスクリプトロジックをサポートしています。
