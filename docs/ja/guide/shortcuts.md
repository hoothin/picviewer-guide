# ショートカットとジェスチャー

## フローティングバーショートカット

デフォルトのショートカットは、フローティングバーが表示されている場合のみ有効です。設定で有効化または変更できます。

| 機能 | デフォルトキー |
| --- | --- |
| 元画像を開く | `A` |
| 元画像を検索 | `S` |
| 現在の画像を開く | `C` |
| 拡大鏡 | `M` |
| ギャラリー | `G` |
| ダウンロード | `D` |

## グローバルプレビューキー

デフォルトで `Ctrl` プレビューモードが有効になっています：

- `Ctrl` を押しながら画像やリンクにホバーすると、大きな画像を直接プレビューできます。
- 放すとプレビューが自動的に閉じます（保持するように設定可能）。

## 操作のヒント

- キーボード操作を好む場合は、「グローバルショートカット」を有効にして、フローティングバーの機能を素早く呼び出すことができます。
- ギャラリーモードは、マウスホイールによるズームと矢印キーによるナビゲーションをサポートしています（サイトやブラウザの設定に依存します）。
