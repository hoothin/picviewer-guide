# Picviewer CE+とは

Picviewer CE+は、Web画像向けの汎用拡張スクリプトであり、「閲覧、選択、収集、エクスポート」のプロセスを可能な限り高速化することを目的としています。

以下の3つの部分が連携して動作します：

- **メインスクリプト**: フローティングバー、プレビュー、ギャラリー、ダウンロード、および全体的なインタラクションを担当します。
- **ルールシステム**: 「サムネイルをどのように元の画像に変換するか」を決定し、遅延読み込み、内部ページ取得、特殊なサイトに対応します。
- **拡張機能**: 空白のギャラリーページとPDFアドオンにより、一括整理と電子アルバム作成をカバーします。

## 動作の概要

1. マウスを画像または認識可能な要素にホバーさせると、スクリプトはルールを照合し、実際の画像アドレスを推測しようとします。
2. 一致に成功すると、フローティングバーが表示され、プレビュー、拡大鏡、ギャラリー、またはダウンロードが可能になります。
3. ギャラリーに入ると、一括フィルタリング、並べ替え、エクスポート、およびページをまたいだ収集が可能になります。

## 適用シーン

- 元画像や大きな画像を素早く確認したいWebサイト
- 画像の遅延読み込みやページネーションがあるギャラリーサイト
- 一括ダウンロード、整理、リネーム、フォーマット変換
- 画像検索やマルチエンジン比較
