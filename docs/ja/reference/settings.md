# 設定一覧

スクリプトの設定は、フローティングバー、拡大鏡、ギャラリー、画像ウィンドウ、およびその他のオプションの4つのカテゴリに分かれています。

## 設定の開き方

スクリプトメニューで「Picviewer CE+ config」を開くか、ギャラリーコマンドで「設定」を選択します。

## カテゴリ概要

### floatBar

フローティングバーの表示、位置、ショートカット、およびグローバルプレビュー動作を制御します。

### magnifier

拡大鏡の半径とマウスホイールのズーム動作を制御します。

### gallery

ギャラリーの収集戦略、表示モード、ダウンロード、およびエクスポート動作を制御します。

### imgWindow

画像ウィンドウのズーム、回転、終了戦略、および背景オーバーレイを制御します。

### other

言語選択、デバッグログ、ルール、検索エンジン設定が含まれます。

> 完全なデフォルト値とフィールドの説明については、「デフォルト値」を参照してください。
