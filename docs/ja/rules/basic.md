# 基本フィールド

基本フィールドは、「サムネイルから元画像へ」の最も一般的な置換を完了するために使用されます。

## name

ルール名。識別とデバッグに使用されます。

## url

サイトマッチングルール。URLが一致した場合にのみルールが有効になります。

```js
url: /^https?:\/\/example\.com\//
```

## src

画像アドレスマッチングルール。特定の画像ドメインやパスを制限するためによく使用されます。

```js
src: /cdn\.example\.com\/thumb\//
```

## r / s

置換ルール。文字列または正規表現をサポートします。`r` はマッチング用、`s` は置換用です。

```js
r: /_thumb/,
s: '_large'
```

配列をサポート：複数の置換が順にマッチングされ、配列の長さは対応している必要があります。

```js
r: [/\/m(\d+)/, /thumb/],
s: ['/l$1', 'large']
```

正規表現置換はグループをサポートします：`$1`、`$2` など。

## enabled

ルールの切り替えに使用されます。

```js
enabled: true
```

## example

サイトの例へのリンク。テストやメンテナンスに便利です。
