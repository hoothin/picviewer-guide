# Settings Overview

Script configuration is divided into four categories: Float Bar, Magnifier, Gallery, Image Window, and Other options.

## How to Open Settings

Open "Picviewer CE+ config" in the script menu, or select "Settings" in the gallery commands.

## Category Overview

### floatBar

Controls float bar display, position, shortcuts, and global preview behavior.

### magnifier

Controls magnifier radius and mouse wheel zoom behavior.

### gallery

Controls gallery collection strategy, display mode, download, and export behavior.

### imgWindow

Controls image window zoom, rotation, close strategy, and background overlay.

### other

Includes language selection, debug logs, rules, and search engine settings.

> For complete default values and field explanations, see "Defaults".
