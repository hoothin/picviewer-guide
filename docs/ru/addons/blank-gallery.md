# Пустая галерея

Пустая страница галереи используется для независимого открытия галереи, подходит для локальной или кросс-сайтовой организации.

## Как открыть

Перейдите по ссылке: https://hoothin.github.io/UserScripts/Picviewer%20CE+/gallery.html

Откройте прямо в браузере, затем перетащите следующее:
- Изображения
- Папки
- Видео или аудио

## Параметры URL

| Параметр | Описание |
| --- | --- |
| `mode` | `1` для режима ViewMore |
| `imgs` | Список URL изображений (разделенных пробелом) |

### Пример

```
https://hoothin.github.io/UserScripts/Picviewer%20CE+/gallery.html?mode=0&imgs=http://xxx/xxx[01-99].jpg
```

> Поддерживает развертывание числового диапазона `[01-99]`.
