# PDF аддон

PDF-плагин заменяет упаковку в ZIP на создание цифрового PDF-альбома.
Подходит для пакетной организации коллекций изображений.

## Ссылка для скачивания

- https://hoothin.github.io/UserScripts/Picviewer%20CE+/pvcep_pdf_addon.user.js

## Как использовать

1. Установите скрипт PDF-плагина.
2. Включите «Сжать в ZIP» в галерее.
3. При экспорте будет создан PDF-файл.

## Типичный сценарий

Когда изображения на сайте пронумерованы последовательно, их можно добавить пакетом в галерею и экспортировать в PDF.
Например, от `1.jpg` до `99.jpg`.

## Примечания

- Плагин зависит от `jsPDF`, при первом использовании может потребоваться время на загрузку.
- Процесс создания PDF вставляет изображения по одному; при большом объеме рекомендуется пакетная обработка.
