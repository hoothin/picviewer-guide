# Скачивание и конвертация

Возможности скачивания сосредоточены в галерее и плавающей панели, поддерживая несколько методов экспорта.

## Способы скачивания

- **Прямое скачивание**: Скачивание одного изображения или пакета на локальный диск.
- **Упаковка в ZIP**: Упаковка в один клик, подходит для больших партий.
- **Удаленная загрузка aria2**: Отправка задач в aria2 RPC.

## Конвертация форматов

Используйте `gallery.formatConversion` для определения правил конвертации, например:

```
webp>png
```

Поддерживает многострочную конфигурацию, попытки сопоставления идут по порядку.

## Конфигурация aria2

| Настройка | Описание | По умолчанию |
| --- | --- | --- |
| `gallery.aria2Host` | Адрес RPC | `http://localhost:6800` |
| `gallery.aria2Token` | Токен RPC | `""` |

## Контроль темпа скачивания

`gallery.downloadGap` используется для управления интервалом между запросами, чтобы избежать блокировок сайта.

## Рекомендуемое использование

- Для пакетной загрузки отдавайте предпочтение ZIP или aria2.
- Для сайтов с большими изображениями рекомендуется использовать конвертацию форматов, чтобы избежать скачивания webp, с которыми может быть неудобно работать.
