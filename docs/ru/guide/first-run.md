# Первый запуск

::: tip Поздравляем с установкой
Ниже — рекомендуемые шаги, чтобы быстро начать работу.
:::

## Следующие шаги

1. **Проверьте работу**: обновите любую страницу, наведите курсор на изображение и убедитесь, что появляется плавающая панель.
2. **Попробуйте основные сценарии**: откройте Галерею через `Ctrl + G`, затем удерживайте `Ctrl` и наведите курсор для быстрого предпросмотра.
3. **Настройте загрузку**: в настройках выберите способ загрузки (ZIP или aria2) и формат по умолчанию.
4. **Дополнительные аддоны**: если нужен экспорт в PDF, установите PDF-аддон на странице аддонов.
5. **Сложные сайты**: при миниатюрах или ленивой загрузке включите или добавьте правила для получения оригиналов.
