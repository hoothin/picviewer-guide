# Плавающая панель

Плавающая панель — это основной элемент управления скрипта, используемый для предпросмотра, открытия оригинала, лупы, галереи и скачивания.

## Условия срабатывания

- Курсор наведен на элемент изображения.
- Или система правил успешно сопоставила не-изображение и вернула доступный URL изображения.

## Основные настройки

| Настройка | Описание | По умолчанию |
| --- | --- | --- |
| `floatBar.butonOrder` | Порядок кнопок | `actual,current,gallery,magnifier,download` |
| `floatBar.showDelay` | Задержка показа (мс) | `100` |
| `floatBar.hideDelay` | Задержка скрытия (мс) | `566` |
| `floatBar.position` | Позиция | `top left` |
| `floatBar.forceShow.enabled` | Принудительный показ | `true` |
| `floatBar.forceShow.size` | Порог размера для принудительного показа | `w:100 h:100` |
| `floatBar.minSizeLimit` | Порог минимального размера | `w:50 h:50` |

## Описание поведения

- **Принудительный показ**: Даже если изображение не масштабировано, панель появляется, если размер соответствует порогу.
- **Минимальный размер**: Слишком маленькие изображения не вызывают панель, чтобы уменьшить случайные нажатия.
- **Фоновые изображения**: Может работать с CSS background-image (переключается в настройках).

## Глобальные горячие клавиши

Если включены глобальные горячие клавиши, можно войти в режим предпросмотра с помощью комбинаций `Ctrl/Alt/Shift/Command`, что подходит для сайтов без явных элементов изображений.
