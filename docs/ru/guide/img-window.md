# Окно изображения

Окно изображения используется для просмотра оригинальных или больших изображений, предоставляя полный набор функций масштабирования, вращения и перетаскивания.

## Основные функции

- Перетаскивание для перемещения, колесико для зума
- Вращение и отражение (настраиваемый шаг)
- Оптимизация отображения длинных изображений

## Основные настройки

| Настройка | Описание | По умолчанию |
| --- | --- | --- |
| `imgWindow.fitToScreen` | По размеру экрана | `true` |
| `imgWindow.fitToScreenSmall` | Маленькие тоже по экрану | `false` |
| `imgWindow.suitLongImg` | Режим длинных изображений | `true` |
| `imgWindow.defaultTool` | Инструмент по умолчанию | `hand` |
| `imgWindow.close.escKey` | Закрыть по Esc | `true` |
| `imgWindow.close.dblClickImgWindow` | Двойной клик для закрытия | `true` |
| `imgWindow.close.clickOutside` | Клик снаружи для закрытия | `""` |
| `imgWindow.shiftRotateStep` | Шаг вращения с Shift | `15` |
| `imgWindow.zoom.range` | Список масштабов | `0.1 ~ 9.0` |

## Советы по использованию

- Удерживайте `Shift` при вращении для точных углов.
- Если возникают конфликты с перекрытием на сайте, можно отключить «Клик снаружи для закрытия».
