# Языки и поддержка многоязычности

Скрипт имеет встроенные языковые пакеты и поддерживает автоматическое определение и ручное переключение.

## Языковой механизм

- Когда `customLang` установлен в `auto`, автоматически определяется язык браузера.
- Также можно вручную указать конкретный язык.

## Пользовательские языковые пакеты

Языковые пакеты основаны на `pvcep_lang.js`, содержащем:

- Список языков
- Сопоставление полей и текста
- Функцию переключения языка

Если нужно добавить новый перевод, можно добавить новую запись языка на основе шаблона.

## Советы

- Если на сайте отсутствует текст, проверьте соответствие `customLang`.
- При настройке языка сохраняйте ключи полными, чтобы избежать пропажи строк.
