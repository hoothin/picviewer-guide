# Лупа

Лупа обеспечивает локальное увеличение, подходящее для проверки деталей и наблюдения на уровне пикселей.

## Основные функции

- Следование за курсором мыши для отображения увеличенной области
- Поддержка масштабирования колесиком мыши
- Настраиваемый диапазон и шаг масштабирования

## Основные настройки

| Настройка | Описание | По умолчанию |
| --- | --- | --- |
| `magnifier.radius` | Радиус лупы | `77` |
| `magnifier.wheelZoom.enabled` | Включить зум колесиком | `true` |
| `magnifier.wheelZoom.pauseFirst` | Пауза перед зумом | `true` |
| `magnifier.wheelZoom.scaleImage` | Масштабировать изображение вместо линзы | `true` |
| `magnifier.wheelZoom.range` | Список масштабов | `0.1 ~ 9.0` |

## Советы по использованию

- Если лупа слишком чувствительна, можно отключить зум колесиком или настроить список масштабов.
- Если следование за курсором неудобно, можно временно включить «Паузу перед зумом».
