# Открыть настройки

::: tip Из-за ограничений страницы открыта эта страница
Окно настроек будет открыто здесь.
:::

## Если окно настроек не появилось

1. Убедитесь, что менеджер пользовательских скриптов включён.
2. Обновите эту страницу один раз.
3. В Chrome 138+ включите **Разрешить пользовательские скрипты** или **Режим разработчика** в расширении менеджера скриптов.
