# Поиск по картинке

Скрипт имеет встроенную поддержку множества поисковых систем, позволяя быстро переключаться между ними без копирования ссылок.

## Движки по умолчанию

Список поиска по умолчанию (можно менять порядок, добавлять/удалять):

- Google
- Yandex
- SauceNAO
- IQDB / 3D IQDB
- Baidu
- Bing
- TinEye
- Sogou
- 360
- Ascii2D
- Trace Moe
- KarmaDecay
- QRCode / QRdecode

## Основные настройки

| Настройка | Описание | По умолчанию |
| --- | --- | --- |
| `firstEngine` | Движок по умолчанию | `Tineye` |
| `gallery.searchData` | Список движков | Предустановленный шаблон |

## Пользовательские поисковые системы

Отредактируйте `gallery.searchData` в настройках, формат:

```
Название движка | Ссылка поиска#t#
```

`#t#` будет заменено на URL изображения.

## Советы по использованию

- Для аниме/иллюстраций рекомендуются SauceNAO и Ascii2D.
- Для фотографий людей отдавайте предпочтение Google и Yandex.
