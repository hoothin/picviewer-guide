# Горячие клавиши и жесты

## Горячие клавиши плавающей панели

По умолчанию работают только при появлении плавающей панели. Можно включить или изменить в настройках.

| Функция | Клавиша по умолчанию |
| --- | --- |
| Открыть оригинал | `A` |
| Найти оригинал | `S` |
| Открыть текущее | `C` |
| Лупа | `M` |
| Галерея | `G` |
| Скачать | `D` |

## Клавиша глобального предпросмотра

Режим предпросмотра по `Ctrl` включен по умолчанию:

- Удерживайте `Ctrl` и наведите курсор на изображение или ссылку для прямого предпросмотра большого изображения.
- Отпустите, чтобы закрыть предпросмотр (можно настроить, чтобы оставался).

## Советы по использованию

- Если вы предпочитаете клавиатуру, включите «Глобальные горячие клавиши» для быстрого доступа к функциям плавающей панели.
- В галерее поддерживается масштабирование колесиком мыши и навигация стрелками (зависит от сайта и настроек браузера).
