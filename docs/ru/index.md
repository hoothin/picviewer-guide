## Что вы можете делать?

- Наведите курсор для предпросмотра любого изображения с быстрым зумом и поворотом.
- В один клик переходите в режим галереи для сбора и управления изображениями на странице.
- Используйте систему правил для адаптации к сложным сайтам и получения оригиналов.
- Ищите по картинке или скачивайте пакетами, создавая полный рабочий процесс.

> Эта документация охватывает установку, настройку, написание правил и практические примеры, подходящие как для новичков, так и для продвинутых пользователей.
