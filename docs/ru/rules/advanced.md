# Продвинутые поля

Продвинутые поля используются для повышения точности правил и улучшения взаимодействия.

## lazyAttr

Укажите имя атрибута ленивой загрузки.

```js
lazyAttr: 'data-original'
```

## exclude

Исключите определенные адреса изображений, чтобы избежать ложных совпадений.

```js
exclude: /weixin_code\.png$/i
```

## description

Поле описания, используемое в галерее, поддерживает селектор или XPath.

```js
description: '.image-caption'
```

## clickToOpen

Используется для указания поведения при клике для открытия.

```js
clickToOpen: {
  enabled: true,
  preventDefault: true,
  type: 'actual'
}
```

## Когда использовать продвинутые поля

- Сайт с ленивой загрузкой: Отдавайте приоритет `lazyAttr`.
- Нужен заголовок или подпись: Используйте `description`.
- Сайт требует клика для открытия большого изображения: Используйте `clickToOpen`.
