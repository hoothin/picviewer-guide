# ext: Захват соседних элементов

`ext` используется для попытки получить изображение из соседних узлов, когда курсор наведен на элемент, не являющийся изображением.

## Поддерживаемые встроенные значения

- `previous`: Предыдущий соседний узел
- `previous-2`: Два предыдущих соседних узла
- `next`: Следующий соседний узел

## Пример

```js
ext: 'previous-2'
```

## Форма функции

```js
ext: function (target) {
  if (target && target.previousElementSibling) {
    return target.previousElementSibling.querySelector('img')
  }
}
```

## Советы

Когда структура страницы стабильна, `ext` может быстро решить задачу получения изображения при «наведении на кнопку или текст».
