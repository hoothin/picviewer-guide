# getExtSrc: Получение из не-изображения

Когда курсор наведен на элемент, не являющийся изображением, можно использовать `getExtSrc` для прямого возврата URL изображения.

## Пример: Поиск изображения рядом с кнопкой

```js
getExtSrc: function () {
  if (this.tagName === 'A' && this.className === 'msk') {
    return this.previousElementSibling.src.replace(/\?param=\d+y\d+/, '')
  }
}
```

## Возвращаемое значение

- `string`: URL одного изображения.
- `array`: URL нескольких изображений.

## Примечания

`getExtSrc` срабатывает только тогда, когда цель не является элементом `img`, подходит для карточек, кнопок или обложек видео.
