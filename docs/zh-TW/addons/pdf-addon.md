# PDF 插件

PDF 插件用於將 ZIP 打包替換為 PDF 電子畫冊。
適合批量整理圖集。

## 下載地址

- https://hoothin.github.io/UserScripts/Picviewer%20CE+/pvcep_pdf_addon.user.js

## 使用方式

1. 安裝 PDF 插件腳本。
2. 在畫廊中啟用「壓縮為 ZIP」。
3. 導出時將直接生成 PDF 文件。

## 典型場景

當網站圖片編號連續時，可在畫廊中批量添加並導出 PDF。
例如 `1.jpg` 到 `99.jpg`。

## 注意事項

- 插件依賴 `jsPDF`，首次使用時可能需要稍等加載。
- PDF 生成過程會逐張插入圖片，體積較大時建議分批處理。
