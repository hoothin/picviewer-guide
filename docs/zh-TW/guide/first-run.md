# 首次運行

::: tip 恭喜安裝成功
以下是建議的下一步操作，幫助你快速進入使用狀態。
:::

## 下一步操作

1. **確認已生效**：重新整理任意網頁，將滑鼠移到圖片上，確認浮動工具列出現。
2. **體驗核心流程**：按 `Ctrl + G` 進入畫廊；按住 `Ctrl` 懸停圖片進行快速預覽。
3. **設定下載方式**：在設定中選擇下載方式（ZIP 或 aria2）並確認預設格式。
4. **可選外掛**：如需匯出 PDF，請到外掛頁安裝 PDF 外掛。
5. **處理特殊站點**：遇到縮圖或延遲載入無法取原圖時，啟用或新增規則。
