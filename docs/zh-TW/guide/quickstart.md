# 快速上手

## 1. 懸停預覽

把鼠標停在圖片上，會出現浮動工具欄。默認包含：原圖、當前圖、畫廊、放大鏡、下載。

## 2. 打開畫廊

按下 `Ctrl + G` 可快速進入畫廊。畫廊會自動收集當前頁面的可識別圖片，並提供篩選、排序與下載功能。

## 3. 快捷預覽

按住 `Ctrl` 懸停圖片或鏈接，可直接預覽大圖。若目標站點較重，建議開啟等待圖片加載以提升穩定性。

## 4. 批量下載

在畫廊中可選擇：

- 導出為 ZIP
- 直接下載圖片列表
- 發送到 aria2

## 5. 規則適配

如果遇到縮略圖或懶加載導致無法獲取原圖，請添加或啟用規則。規則系統支持 URL 替換、內頁抓取與自定義腳本邏輯。
