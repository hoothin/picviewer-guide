# 快捷鍵與手勢

## 浮動工具欄快捷鍵

默認快捷鍵僅在浮動工具欄出現時有效，可在設置中開啟或修改。

| 功能 | 默認鍵位 |
| --- | --- |
| 打開原圖 | `A` |
| 搜尋原圖 | `S` |
| 打開當前圖 | `C` |
| 放大鏡 | `M` |
| 畫廊 | `G` |
| 下載 | `D` |

## 全局預覽鍵

默認開啟 `Ctrl` 預覽模式：

- 按住 `Ctrl` 懸停圖片或鏈接，直接預覽大圖
- 釋放後自動關閉預覽（可配置為保持）

## 常用操作建議

- 習慣用鍵盤時可開啟「全局快捷鍵」，快速調用浮動工具欄功能
- 畫廊中支持滾輪縮放與方向鍵瀏覽（取決於站點與瀏覽器設置）
