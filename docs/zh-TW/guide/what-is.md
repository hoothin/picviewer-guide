# 腳本是什麼

Picviewer CE+ 是一個面向網頁圖片的通用增強腳本，目標是讓「看圖、選圖、收集、導出」的過程盡量快。

它由三部分協同：

- **主腳本**：負責浮動工具欄、預覽、畫廊、下載與全局交互。
- **規則系統**：決定「如何把縮略圖變成原圖」，並兼容懶加載、內頁抓取與特殊站點。
- **擴展組件**：空白畫廊頁與 PDF 插件，覆蓋批量整理與電子畫冊。

## 工作方式概覽

1. 當鼠標懸停到圖片或可識別的元素時，腳本嘗試匹配規則並推斷真實圖片地址。
2. 匹配成功後，浮動工具欄出現，可進行預覽、放大鏡、畫廊或下載。
3. 進入畫廊後，可進行批量篩選、排序、導出與跨頁收集。

## 適用場景

- 需要快速查看原圖或大圖的網站
- 圖片懶加載或跨頁加載的圖庫站點
- 批量下載、整理、重命名與格式轉換
- 以圖搜圖或多引擎對比
