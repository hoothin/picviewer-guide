# 設置項總覽

腳本配置分為四大類：浮動工具欄、放大鏡、畫廊、圖片窗，以及通用選項。

## 如何打開配置

在腳本菜單中打開「Picviewer CE+ config」，或在畫廊命令中選擇「打開設置」。

## 分類概覽

### floatBar

控制浮動工具欄顯示、位置、快捷鍵與全局預覽行為。

### magnifier

控制放大鏡半徑與滾輪縮放行為。

### gallery

控制畫廊的收集策略、顯示模式、下載與導出行為。

### imgWindow

控制圖片窗的縮放、旋轉、關閉策略與背景遮罩。

### other

包含語言選擇、調試日誌、規則與搜尋引擎設置。

> 完整默認值與字段解釋見「默認值速查」。
