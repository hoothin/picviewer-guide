# 基礎字段

基礎字段用於完成「從縮略圖到原圖」的最常見替換。

## name

規則名稱，用於識別與調試。

## url

站點匹配規則。只有 URL 命中時，規則才會生效。

```js
url: /^https?:\/\/example\.com\//
```

## src

圖片地址匹配規則。常用於限定特定圖片域名或路徑。

```js
src: /cdn\.example\.com\/thumb\//
```

## r / s

替換規則，支持字符串或正則。`r` 用於匹配，`s` 用於替換。

```js
r: /_thumb/,
s: '_large'
```

支持數組：多條替換按順序匹配，數組長度需對應。

```js
r: [/\/m(\d+)/, /thumb/],
s: ['/l$1', 'large']
```

正則替換支持分組：`$1`、`$2` 等。

## enabled

用於開關規則。

```js
enabled: true
```

## example

站點示例鏈接，便於測試與維護。
