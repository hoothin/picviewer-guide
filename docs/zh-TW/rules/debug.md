# 排錯與調試

規則調試的目標是確認「規則是否命中」和「返回的 URL 是否正確」。

## 開啟調試

在設置中啟用 `debug`，瀏覽器控制台會輸出更多匹配信息。

## 排錯步驟

1. 確認 `url` 是否匹配目標站點。
2. 檢查 `src` / `lazyAttr` 是否命中。
3. 核對 `r/s` 順序與正則。
4. 使用 `xhr` 時驗證選擇器與鏈接。
5. 複雜邏輯在 `getImage` / `getExtSrc` 內加日誌。

## 常見問題

- 替換後仍是縮略圖：`r/s` 匹配不足或順序錯誤。
- 規則未生效：`url` 過寬或過窄。
- 內頁抓取失敗：選擇器未更新或跨域限制。
