# PDF 插件

PDF 插件用于将 ZIP 打包替换为 PDF 电子画册。
适合批量整理图集。

## 下载地址

- https://hoothin.github.io/UserScripts/Picviewer%20CE+/pvcep_pdf_addon.user.js

## 使用方式

1. 安装 PDF 插件脚本。
2. 在画廊中启用“压缩为 ZIP”。
3. 导出时将直接生成 PDF 文件。

## 典型场景

当网站图片编号连续时，可在画廊中批量添加并导出 PDF。
例如 `1.jpg` 到 `99.jpg`。

## 注意事项

- 插件依赖 `jsPDF`，首次使用时可能需要稍等加载。
- PDF 生成过程会逐张插入图片，体积较大时建议分批处理。
