# 首次运行

::: tip 恭喜安装成功
下面是建议的下一步操作，帮助你快速进入使用状态。
:::

## 下一步操作

1. **确认已生效**：刷新任意网页，将鼠标移到图片上，确认浮动工具栏出现。
2. **体验核心流程**：按 `Ctrl + G` 进入画廊；按住 `Ctrl` 悬停图片进行快速预览。
3. **设置下载方式**：在设置中选择下载方式（ZIP 或 aria2）并确认默认格式。
4. **可选插件**：如需导出 PDF，请到插件页安装 PDF 插件。
5. **处理特殊站点**：遇到缩略图或懒加载无法取原图时，启用或新增规则。
