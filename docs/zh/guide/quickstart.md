# 快速上手

## 1. 悬停预览

把鼠标停在图片上，会出现浮动工具栏。默认包含：原图、当前图、画廊、放大镜、下载。

## 2. 打开画廊

按下 `Ctrl + G` 可快速进入画廊。画廊会自动收集当前页面的可识别图片，并提供筛选、排序与下载功能。

## 3. 快捷预览

按住 `Ctrl` 悬停图片或链接，可直接预览大图。若目标站点较重，建议开启等待图片加载以提升稳定性。

## 4. 批量下载

在画廊中可选择：

- 导出为 ZIP
- 直接下载图片列表
- 发送到 aria2

## 5. 规则适配

如果遇到缩略图或懒加载导致无法获取原图，请添加或启用规则。规则系统支持 URL 替换、内页抓取与自定义脚本逻辑。
