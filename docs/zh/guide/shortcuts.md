# 快捷键与手势

## 浮动工具栏快捷键

默认快捷键仅在浮动工具栏出现时有效，可在设置中开启或修改。

| 功能 | 默认键位 |
| --- | --- |
| 打开原图 | `A` |
| 搜索原图 | `S` |
| 打开当前图 | `C` |
| 放大镜 | `M` |
| 画廊 | `G` |
| 下载 | `D` |

## 全局预览键

默认开启 `Ctrl` 预览模式：

- 按住 `Ctrl` 悬停图片或链接，直接预览大图
- 释放后自动关闭预览（可配置为保持）

## 常用操作建议

- 习惯用键盘时可开启“全局快捷键”，快速调用浮动工具栏功能
- 画廊中支持滚轮缩放与方向键浏览（取决于站点与浏览器设置）
