# 脚本是什么

Picviewer CE+ 是一个面向网页图片的通用增强脚本，目标是让"看图、选图、收集、导出"的过程尽量快。

它由三部分协同：

- **主脚本**：负责浮动工具栏、预览、画廊、下载与全局交互。
- **规则系统**：决定"如何把缩略图变成原图"，并兼容懒加载、内页抓取与特殊站点。
- **扩展组件**：空白画廊页与 PDF 插件，覆盖批量整理与电子画册。

## 工作方式概览

1. 当鼠标悬停到图片或可识别的元素时，脚本尝试匹配规则并推断真实图片地址。
2. 匹配成功后，浮动工具栏出现，可进行预览、放大镜、画廊或下载。
3. 进入画廊后，可进行批量筛选、排序、导出与跨页收集。

## 适用场景

- 需要快速查看原图或大图的网站
- 图片懒加载或跨页加载的图库站点
- 批量下载、整理、重命名与格式转换
- 以图搜图或多引擎对比
