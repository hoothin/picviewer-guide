# 设置项总览

脚本配置分为四大类：浮动工具栏、放大镜、画廊、图片窗，以及通用选项。

## 如何打开配置

在脚本菜单中打开“Picviewer CE+ config”，或在画廊命令中选择“打开设置”。

## 分类概览

### floatBar

控制浮动工具栏显示、位置、快捷键与全局预览行为。

### magnifier

控制放大镜半径与滚轮缩放行为。

### gallery

控制画廊的收集策略、显示模式、下载与导出行为。

### imgWindow

控制图片窗的缩放、旋转、关闭策略与背景遮罩。

### other

包含语言选择、调试日志、规则与搜索引擎设置。

> 完整默认值与字段解释见“默认值速查”。
