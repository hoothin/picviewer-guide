# 基础字段

基础字段用于完成"从缩略图到原图"的最常见替换。

## name

规则名称，用于识别与调试。

## url

站点匹配规则。只有 URL 命中时，规则才会生效。

```js
url: /^https?:\/\/example\.com\//
```

## src

图片地址匹配规则。常用于限定特定图片域名或路径。

```js
src: /cdn\.example\.com\/thumb\//
```

## r / s

替换规则，支持字符串或正则。`r` 用于匹配，`s` 用于替换。

```js
r: /_thumb/,
s: '_large'
```

支持数组：多条替换按顺序匹配，数组长度需对应。

```js
r: [/\/m(\d+)/, /thumb/],
s: ['/l$1', 'large']
```

正则替换支持分组：`$1`、`$2` 等。

## enabled

用于开关规则。

```js
enabled: true
```

## example

站点示例链接，便于测试与维护。
